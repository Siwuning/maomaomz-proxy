## 🎮 使用流程

### 第1步：输入信息
1. 输入想要占卜的角色名字
2. 选择一种占卜类型
3. 点击"开始占卜"

### 第2步：选择塔罗牌
- 从3张牌中选择1张
- 点击卡片进行翻牌
- 观看翻牌动画

### 第3步：查看结果
- 自动显示占卜结果
- 带有神秘的描述文本
- **4种操作方式：**
  1. **复制结果** - 复制纯文本
  2. **发送到聊天** - 直接发送到 SillyTavern 聊天框
  3. **创建快速回复** - 生成可复用的 QR 按钮
  4. **重新占卜** - 清空重新开始

## ⚡ 快速回复（QR）功能

### 什么是快速回复？
快速回复（Quick Reply）是 SillyTavern 的功能，可以创建**可复用的按钮**，点击后自动发送预设消息。

### 如何使用？

#### 方式一：自动创建（推荐）
1. 完成占卜后，点击 **"创建快速回复"** 按钮
2. 如果在 SillyTavern 中，会自动创建 QR 按钮
3. 在聊天框上方会出现占卜结果按钮

#### 方式二：手动创建
1. 点击 **"创建快速回复"** 后，会复制 QR 命令
2. 粘贴到 SillyTavern 聊天框
3. 发送命令，即可创建按钮

### QR 命令格式
```
/qr create label="🔮 角色名 占卜类型" 占卜结果内容
```

### 使用场景
- ✅ **保存有趣的占卜结果**，随时重新发送
- ✅ **分享给角色**，让角色看到占卜结果
- ✅ **创建剧情按钮**，丰富对话选项
- ✅ **快速测试**，不用每次重新占卜

## 🎨 自定义建议

### 增加新的占卜类型
在 `script.js` 的 `divinationResults` 对象中添加：

```javascript
'new-type': {
  title: '新类型占卜',
  results: [
    {
      icon: '🌟',
      text: '占卜结果文本，使用 {name} 作为角色名占位符'
    },
    // ... 更多结果
  ]
}
```

### 修改结果文本
直接编辑 `script.js` 中对应类型的 `results` 数组即可。

### 调整配色
修改 `style.css` 中的 `:root` CSS 变量。
