# 🎮 角色创建系统 - 像素科技版

**暗色像素科技风角色创建工具**

---

## ✨ 特性

### 🎨 视觉设计
- ✅ **纯暗色背景**（绝对无渐变色）
- ✅ **像素科技风格**（网格背景 + 扫描线效果）
- ✅ **霓虹边框**（赛博绿/蓝/紫配色）
- ✅ **Courier New 等宽字体**（复古科技感）
- ✅ **方形界面元素**（像素化设计）
- ✅ **发光效果**（Box-shadow 霓虹光）

### 🚀 功能特性
- ✅ **5步引导流程**（基本信息 → 外貌 → 性格 → 背景 → 技能）
- ✅ **实时头像预览**
- ✅ **属性值滑块**（动态颜色变化）
- ✅ **性格标签选择器**
- ✅ **技能列表管理**（添加/删除）
- ✅ **自动草稿保存**（7天有效）
- ✅ **键盘快捷键**（Ctrl+S 保存，Ctrl+← → 切换步骤）
- ✅ **多格式导出**（JSON / YAML）
- ✅ **响应式设计**（移动端优化）

### ⌨️ 快捷键
- `Ctrl + S` - 保存草稿
- `Ctrl + ←` - 上一步
- `Ctrl + →` - 下一步
- `Ctrl + Enter` - 生成角色卡（在第5步）

---

## 📋 字段说明

### 第1步：基本信息
- 名字（必填）
- 年龄、性别
- 种族、职业
- 称号/绰号
- 头像链接

### 第2步：外貌特征
- 身高、体重、体型
- 发色、发型、瞳色
- 特殊特征
- 常穿服装

### 第3步：性格特点
- 性格类型（多选标签）
- 性格描述
- 喜好、厌恶、恐惧

### 第4步：背景故事
- 出生地
- 家庭背景
- 成长经历
- 行动动机
- 秘密

### 第5步：技能能力
- 属性值（力量/敏捷/智力/魅力/幸运/意志）
- 技能列表
- 特殊能力
- 弱点

## 🎯 使用建议

1. **头像链接**：建议使用直链图片，支持实时预览
2. **属性值**：根据角色定位合理分配，总和无限制
3. **技能列表**：等级范围 1-10，建议不超过 8 个技能
4. **文本描述**：支持多行输入，建议详细描述
5. **草稿保存**：定期使用 Ctrl+S 手动保存
